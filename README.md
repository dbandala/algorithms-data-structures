# algorithms-data-structures
Collection of implementations for fundamental algorithms, data structures, and solutions to common problems encountered in technical interviews. Includes a complete embedded systems foundations course for senior embedded engineer interview preparation.

## Contents

| Folder | Description |
|--------|-------------|
| `algorithms/` | Classical algorithms in Python and C++ |
| `data-structures/` | Core data structure implementations in Python |
| `problem-solving/` | LeetCode-style problem solutions |
| `interview-problems/` | Company-specific interview problems |
| `sorting-algorithms/` | Sorting algorithm implementations |
| `system-design/` | System design study notes |
| `embedded_systems/` | **Complete embedded systems course** — C/C++, RTOS, hardware interfaces, interview prep |
